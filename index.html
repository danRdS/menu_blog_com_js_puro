Percorrer elementos

Veremos neste pequeno artigo algumas maneiras de se percorrer elementos filhos de um elemento pai em Javascript. Isto é muito útil para poder realizar ações específicas de forma individual ou conjunta em cada um dos elementos filhos de forma dinâmica dependendo da interação do usuário com eles. Vamos a um trecho de código HTML para iniciarmos nossa exemplificação:

Neste exmplo, foi formado um pequeno menu (criado com a tag ul) no qual contém 4 elementos do tipo li. Cada um deles contém um evento de clique (onclick), chamado "mudarCor" enviando o elemento alvo do clique como parâmetro desta função para que através dele seja possível saber qual dos elementos foi clicado para realizar qualquer que seja a reação ao clique daquele evento. Em nosso caso, irá fazer a estilização dele.

Existem algumas formas de percorrer elementos filhos de um elemento pai, dentre elas:

Pegando o elemento pai e o percorre com o Object.values() pra encontrar seus filhos. Dessa foma será possível ter acesso a todas as propriedades dos elementos filhos.

const lista = document.getElementById('lista'); const opcoesLista = lista.children; Object.values(opcoesLista).forEach(opcao => { console.log(opcao); })

Pega o primeiro elemento e o chama por intermédio de uma função recursiva que percorre os próximos elementos com iteração:

const primeiraOpcaoLista = document.querySelector('li'); let index = 0; const apresentarOpcoes = elemento => { console.log(elemento); index = (index + 1) % opcoesLista.length; if(index) { const proximo = elemento.nextElementSibling; apresentarOpcoes(proximo); } } apresentarOpcoes(primeiraOpcaoLista);
